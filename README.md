# Wohnungsgenehmigungs-Kalkulator für Berlin
**Hilft Menschen mit Bürgergeld/Sozialamt, schneller eine passende Wohnung zu bekommen.**

Dieses Tool berechnet, ob eine Miete (Bruttokaltmiete) voraussichtlich von Jobcenter oder
Sozialamt übernommen werden kann — basierend auf den Berliner Richtwerten ab 01.10.2023.

### Was das Tool kann
- Nettokaltmiete, kalte Nebenkosten & Heizkosten eingeben
- Richtwert automatisch prüfen
- Klimabonus & Zuschläge (Härtefall, Wohnungslosigkeit etc.)
- Ergebnis sofort verständlich dargestellt
- Funktioniert auf Handy, Tablet, PC
- Sprachen: **Deutsch**, **Русский**, **Українська**

### Warum dieses Tool wichtig ist
Menschen, die neu in Berlin sind, stehen oft vor bürokratischen Hürden.
Dieses Tool gibt Orientierung — **schnell & kostenlos**.

### Nutzung
Dieses Projekt ist Teil von „Antrag-Hilfe Berlin“, einem unabhängigen
Angebot zur Unterstützung bei Bürgergeld-, Wohnungs- und Sozialamtsanträgen.
